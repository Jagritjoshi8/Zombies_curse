## Zombies_curse is a React App

![Image](githubimg.png)

This is a UI oriented Zombies list App with Search Filter by Name Functionality.

All Zombies are dangerous with a tragic flashback story and clan is Zombies group with which each zombie is associated.

This web-app is fully responsive, i.e.. you can get scared from Zombies in any device.

## Run following Scripts

In the project directory, you can run:

### `npm start`

### `npm test`

### `npm run build`
